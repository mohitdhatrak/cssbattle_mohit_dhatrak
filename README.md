### Instructions
To run the project locally, follow these steps:
- Clone this repository to your device
- Install node modules using command:
  - for npm: `npm install`
  - for yarn: `yarn install`
- Start the react app using command:
  - for npm: `npm start`
  - for yarn: `yarn start`
